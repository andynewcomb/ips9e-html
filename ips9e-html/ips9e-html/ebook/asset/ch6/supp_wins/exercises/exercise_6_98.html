6.98 Do you agree? State whether or not you agree with each of the following statements and provide a short summary of the reasons for your answers.

  1. (a) If the P-value is larger than 0.05, the null hypothesis is true.

  2. (b) Practical significance is not the same as statistical significance.

  3. (c) We can perform a statistical analysis using any set of data.

  4. (d) If you find an interesting pattern in a set of data, it is appropriate to then use a significance test to determine its significance.

  5. (e) It’s always better to use a significance level of α = 0.05 than to use α = 0.01 because it is easier to find statistical significance.
