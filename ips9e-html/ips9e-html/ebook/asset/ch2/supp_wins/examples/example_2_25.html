EXAMPLE 2.25

Fidgeting and fat gain. Figure 2.20 is a scatterplot of the fidgeting and fat gain data described in Example 2.19 (page 107). There is a negative linear relationship. The two lines on the plot are the two least-squares regression lines. The regression line using nonexercise activity to predict fat gain is blue. The regression line using fat gain to predict nonexercise activity is red. Regression of fat gain on nonexercise activity and regression of nonexercise activity on fat gain give different lines. In the regression setting, you must decide which variable is explanatory.
